# Dashboard Financiero MX

## Descripción

Dashboard Financiero MX es una aplicación web desarrollada para el análisis de activos financieros utilizando indicadores de riesgo y rendimiento.

El sistema permite visualizar información histórica de acciones, métricas actuariales y simulaciones financieras para apoyar la toma de decisiones.

## Funcionalidades

* Consulta de activos financieros.
* Visualización de precios históricos.
* Cálculo de Valor en Riesgo (VaR).
* Cálculo de Expected Shortfall (ES).
* Simulación Monte Carlo.
* Panel de noticias financieras.
* Cartera de inversión simulada.
* Asistente financiero interactivo.

## Tecnologías utilizadas

* HTML
* JavaScript
* Node.js
* Express
* MySQL
* ApexCharts
* Tailwind CSS
